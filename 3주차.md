<개인 탐구>

mask 란 무엇인가?
: annotation 에서의 좌표 값을 바탕으로 하여 실제 목표하는 부분을 표기하는 것

mask 가 필요한 이유
: 실제 학습 전 인간의 구분이 필요하다. 모델을 학습시키는 것을 목적으로 하여 필요한 정보만 표기하는 행위. 불필요한 정보를 지우고 원하는 목표만 인식시킴으로 인해 모델의 학습을 원활하게 한다.
